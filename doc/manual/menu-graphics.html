Gens/GS - Menus - Graphics Menu

Index → Menus → Graphics Menu

The Graphics Menu allows you to adjust Gens/GS's graphical output.

Full Screen: Activate Full Screen mode. (On Linux, the menu system is not accessible in Full Screen mode due to GTK+ limitations.)

VSync: Enable vertical synchronization. This option is only available when using OpenGL (Linux) or DirectDraw (Windows).

Stretch: Set the stretch mode option. Stretch can be set to either "None", "Horizontal Only", "Vertical Only", or "Full Screen". This option is only available when using OpenGL (Linux) or DirectDraw (Windows).

Backend: Selects a video rendering backend. Backends currently implemented:

  • Linux:
    • SDL
    • SDL+OpenGL
  • Windows:
    • DirectDraw 4
    • GDI

OpenGL Linear Filter: Enables linear filtering in OpenGL mode, which can improve visual quality at higher resolutions. (Linux only)

OpenGL Resolution: Sets the rendering resolution for OpenGL mode. (Linux only)

Bits per pixel: Sets the color depth to be used by SDL and SDL+OpenGL. (The Windows version uses the system color depth setting.)

Color Adjust...: Allows you to adjust color options, including brightness and contrast.

Render: Contains a list of all available renderers. The built-in renderers are at the top of the list, followed by renderers found in all loaded MDP plugins.

Sprite Limit: If enabled, emulates the hardware sprite limit found in the original Sega Genesis VDP.

Frame Skip: Specify the number of frames to skip. This is useful on slower machines.

Screen Shot: Saves a screenshot of the current image in PNG format. The screenshot is unfiltered.
