Gens/GS - Menus - Options Menu

Index → Menus → Options Menu

The Options Menu allows you to adjust various options.

General Options: Contains various options, including on-screen display settings, pause tint, and intro effect styles.

Joypads: Allows you to reconfigure the controller types and mappings.

Directories: Configure what directories are used for different types of files, such as savestates or screenshots.

BIOS/Misc Files: Configure different firmware and miscellaneous files, such as SegaCD firmware and the path to the "rar" binary.

Enable SRAM: If checked, SRAM is enabled for games that support it. Some games do not usually have SRAM, and will activate a copy-protection function if they detect it. For those games, you will need to manually disable SRAM by unchecking this item.

Current CD Drive: Select the CD-ROM drive to use for physical SegaCD support via File, Boot CD.

SegaCD SRAM Size: Specify the size for the SegaCD backup RAM cartridge. Available sizes are None, 8 KB, 16 KB, 32 KB, and 64 KB.

Load Config...: Load an alternate configuration file.

Save Config As...: Save the current configuration to the specified file.
