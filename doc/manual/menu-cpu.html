Gens/GS - Menus - CPU Menu

Index → Menus → CPU Menu

The CPU Menu allows you to control the emulated CPUs. Note that most of the functionality in this menu requires a ROM to be loaded.

Debug: Opens the built-in Gens debugger for a specific component. Components include the 68000, Z80, and SH2 CPUs, as well as the VDPs. Note that only components available for the loaded ROM will be displayed. For example, if a 32X ROM is loaded, then only the main Genesis components and the 32X components will be listed.

Country: Sets a specific country code for the emulated system. You can also change the country code order for auto-detection.

Hard Reset: Performs a hard reset of the emulator.

Reset [CPU]: Resets the specified CPU. Only CPUs that are available for the loaded ROM will be displayed.

SegaCD Perfect Sync (SLOW): Enables perfect synchronization for SegaCD emulation. This is slower than regular SegaCD emulation, but is more accurate.
